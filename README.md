# youth-unemployment-project
Youth Unemployment Analysis in Canada (2020–Present)
This project analyzes youth unemployment trends in Canada from 2020 to the present using Power BI dashboards, Python scripts, and CSV datasets. It focuses on individuals aged 15–24 and compares their employment outcomes with adults aged 25 and older.
Youth vs Adult Unemployment Rates: Year-over-year comparison since 2020
Gender Disparities: Differences in unemployment rates between young males and females
Regional Breakdown: Unemployment rates across Canadian provinces
Occupational Trends: Employment distribution among youth by job category
